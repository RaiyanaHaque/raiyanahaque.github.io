Sugar Ball!

Sugar Ball is the first game that I have ever made. This is a simple 3D roller ball game with the objective of getting all of the sugar cubes to finish the game (I did not have any other idea at that moment). The game has a pastel colour scheme and looks quite cute compared to the newer and recent games that I have made. This game gave me the basic skill in Unity as well as pixel art and it started off my journey in C#! Although the game still has major bugs where the killbox does not work but it is still fun to play and make sure not to fall down.

  • First indivdual project
  • Made with Unity; 3D modelling done with Maya; art assets using Photoshop
