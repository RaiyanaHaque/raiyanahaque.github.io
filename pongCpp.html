Pong Game using DirectX Framework

This project was my introduction to using C++ and looking into the basics of the language. I found a lot of similarities with C# which helped me to understand some of the key concepts as well. I made two version of the project, I made one using the console and the other one was created using Chilli's DirectX Framework. I took 3 weeks to create a good basic, playable game and it also gave me the chance to try out the language. I mainly looked at using IF functions and I also learned how to find overlapping areas of two objects as well as how to find the corners of an object.
